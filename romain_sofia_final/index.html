About this Experiment

This project explores procedural generation in real-time using ThreeJS WebGPU and noise algorithms. The terrain evolves dynamically and demonstrates how mathematics can generate natural looking environments without any prebuilt assets.

Everything you see is generated live in the browser. No models, no textures, just code and a lot of GPU power.

Technical Stack

  • ThreeJS WebGPU Renderer
  • Procedural Terrain Generation
  • Perlin / Simplex Noise
  • Real-time GPU Rendering
  • Custom Shader Experiments
